Getting Started

To get started with EduLearn Angular, follow the steps below:

  1. Download the template from the BootstrapDash website
  2. Extract the downloaded file
  3. Open the extracted folder in your code editor
  4. Run the following command to install the dependencies: 
    5. npm install
  5. Run the following command to start the development server: 
    6. ng serve
  6. Open your browser and go to http://localhost:4200
